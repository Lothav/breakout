# Breakout

![Alt Text](https://github.com/Luiz0tavio/breakout/blob/master/bin/data/breakout_gif.gif)


### Dependencies

Linux OS based.

Libs:
- SDL2
- SDL2_image
- GL
- GLEW
- freetype2

### Running

```
$ make
$ cd bin
$ ./breakout
```

### Gameplay


mouse : paddle movement

**q** : quit

**r** : restart

**left click** : pause

**right click** : debug
